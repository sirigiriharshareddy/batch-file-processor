# Batch File Processor - Frontend

This folder contains the React with Vite frontend for the Batch File Processor application.

## Prerequisites

- Node.js (v14+)
- npm

## Setup and Running

1. **Navigate to the frontend directory:**
   ```bash
   cd frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

The application will be accessible at `http://localhost:5173` (default Vite port).

## Project Structure

- `src/`: React source code (components, styles).
- `public/`: Static assets.
- `index.html`: Main HTML file.
- `vite.config.js`: Vite configuration.
- `package.json`: Project metadata and dependencies.
